标注文件管理



标注文件的查看与下载

管理员可以进行标注文件的查看与下载

进入管理界面，点击标注数据（简要）即可查看标注文件的标注用户，标注日期，以及文件大小

示意图如下：

另外：超级管理员可以查看所有的用户的标注信息，并且系统整合了所有用户的标注信息，并提供了下载按钮，对应上图中的all user annotation

选择需要下载的文件，即可下载标注文的excel



进入管理界面，点击标注数据（详细）即可预览标注文件的excel文件

超级管理员可以直接预览all user annotation对应的excel

一般管理员可以预览当前用户最新标注文件的excel形式

excel表中额外根据出生日期和拍片日期计算出了牙龄year_age && month_age && day_age

并且对牙位和牙评分以表格的形式呈现，未标注即为NAN
